Taskwarrior 的使用

常用指令

使用 task 调用 taskwarrior,默认执行 next 指令.

  • next 按照计算出的优先级列出任务及其 ID.

  • ID done 完成 ID 对应的任务.

  • ID delete 删除 ID 对应的任务.

  • add 添加任务.modify 修改任务.尽可能使用 feature:value 语法提供详细的属性或使用 +tagname 语法添加标记,可用的属性列表如下:

    • project 任务所属的项目.

    • due 任务的到期时间

    • priority 任务的优先级, L,M,H 表示 low,mid,high.

    • depends 前置任务的 ID.

  • config 设置,可用的设置项如下:

    • urgency.user.(tag/project).name.coefficient value 设置 tag,project 对任务重要程度的影响.

常用筛选器

可以使用筛选器指定要操作的任务范围.使用 and 连接多个筛选器.

  • /word/ 在描述和注释中出现 word.

  • (entry/start/end)[.before/after]:time\(\pm\)(duration) 添加/开始/结束时间,可以使用一般英文描述或 ISO 8601 标准表示法及其简化.

  • (project/tag):name 项目和标签名称,可留空.

  • status:status 当前状态
