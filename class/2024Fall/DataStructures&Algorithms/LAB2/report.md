### 03.cpp

最严重错误类型: Time Limit Exceeded

原因: 每次使用 `erase` 执行删除的时间复杂度是关于字符串长度线性 $O(n)$ 的,在执行反复多次在字符串头部进行的删除时,时间复杂度达到最坏 $O(nm)$ 超时.

测例:

```
AABBAABBAABB...
500000
0 A
0 A
...
0 A
```

标准答案:

```
ABBAABB...
```

构造思路:保持字符串长度在 $n$ 级别的同时在前端反复进行插入和删除.
