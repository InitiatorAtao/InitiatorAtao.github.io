第五课-よろしくお願いします

(名词句) 相当于汉语的 "...是/不是..."

  • ...は...です(でした) : ...是...(过去式)

  • ...は...ではありません(+でした) : ...不是...(过去式)

  • ...は...で... : ...是...,也是...(以最后的时态和肯否定为准)

  • ...は...でしょう : ...应该是...(表示推量)

(动词句,ます型) 主要由主语+は(+宾语+を)+动词的变形构成,其中接宾语的动词称他动词,否则称自动词,动词在句中变为表示礼貌的ます型,可能的变形如下:

  • 五段动词(以う段假名结尾):将结尾假名变为い段+ます

  • 一段动词(以い,え段假名+る结尾):将结尾る去掉+ます

  • さ变动词(仅する):变为します

  • か变动词(仅くる):变为きます

ます部分的变形可能如下:

  • ...ます(でした)(+か) : 肯定形式(过去式)(询问)

  • ...ました(ませんでした) : 否定形式(过去式)

(指示代词,こそあど 系词语) こ,そ,あ,ど 修饰代词的位置,分别表示离说话人近,离听话人近,离两人都远和未知的位置.

可能的后缀和代表的含义如下:

  • れ 代指事物

  • こ 代指场所

  • ちら 代指方向,或人的敬称

  • の 代指事物,连体词

  • んな 代指某种性质或状态,连体词

  • んなに 代指状态,副词

助词一览

  • の 连接名词,表示 "的"

  • は 表示主语或强调

  • か 在句尾表示疑问

  • を 表示宾语
