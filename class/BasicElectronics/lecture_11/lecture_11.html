一阶电路求解

\[\begin{aligned} f\left( t \right) =&f\left( \infty \right) +Ae^{-\frac{t}{\tau}}\nonumber \end{aligned}\] 求稳态值 \(f\left( \infty \right)\),初始值 \(f\left( 0^{+} \right)\),则 \(A=f\left( 0^{+} \right) -f\left( \infty \right) \big|_{0^{+}}\),从 \(L\) 或 \(C\) 两端求入端等效电阻 \(R_{\texttt{eq}},\tau=R_{\texttt{eq}}C\) 或 \(\frac{L}{R_{\texttt{eq}}}\).

一阶电路的全响应

(全响应) 非零初始状态的电路受到激励时电路中产生的响应.

(全响应及其两种分解方式)

  • 全解=强制分量(稳态解)+自由分量(暂态解) \[\begin{aligned} f\left( t \right) =&f\left( \infty \right) +\left( f\left( 0^{+} \right) -f\left( \infty \right) \right) e^{-\frac{t}{\tau}}\nonumber \end{aligned}\]

  • 全响应=零状态响应 (zero-state response,ZSR)+零输入响应 (zero-input response,ZIR) \[\begin{aligned} f\left( t \right) =&f\left( \infty \right) \left( 1-e^{-\frac{t}{\tau}} \right)+f\left( 0^{+} \right) e^{-\frac{t}{\tau}}\nonumber \end{aligned}\]

    零输入响应:独立源激励为零,由初始储能激励产生响应.线性电路中和初值成正比.

    零状态响应:储能元件无初始储能,由电源激励产生响应.线性电路中和激励成正比.

模拟电子电路基础

CMOS 模拟电路的基本元器件

  • Diode (二极管)

    单向导通,正向电流随电压指数增长 \(I=I_{S}\exp \frac{V_{D}}{V_{T}}\),可视作导线,反向电流 \(I\rightarrow -I_{S}\) 极小,可视作开路.

  • MetaI-Oxide-Semiconductor Field EffectTransistor (MOSFET)

    • NMOS (电子/Negative 负载流子)

    • PMOS (空穴/Positive 正载流子)

    分为 Source (源)、 Drain (漏)、 Gate (栅)、 Body (体).

    阈值电压 \(V_{\texttt{th}}\) 为开启所需施加在 \(GS\) 上的电压.

  • Resistor (电阻)

  • Capacitor (电容)

  • lnductor (电感)

  • Varactor (变容管)
