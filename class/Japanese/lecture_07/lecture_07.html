第七课-日本の四季

(形容词句) 使用形容词修饰对象,句中的く表示形容词的く型,即将い改为く:

  • ...は...です(かったです) : ...有...的特点(过去式)

  • ...は...く+ないです(ありません) : ...没有...的特点(另一种表达)

  • ...は...く+なかったです(ありませんでした) : ...过去没有...的特点(另一种表达)

(形容词活用型) 在不同位置使用的形容词的不同词尾:

  • い 基本型,连体型,后接修饰的名词

  • く 连用型,接否定句式

  • くて 中顿型,继续接形容词

  • ければ 假定型,表示假定条件

  • かろう 推量型,表示推测

(比较句) 表示两者相比或最高性质:

  • ...は...より...です : ...比...更...

  • ...より...のほうが...です : ...不如...更...

  • ...は...ほど...く+ないです(ありません) : ...不如...更...

  • ...ほど...はありません : ...是最...的

助词一览

  • より 表示比较的对象

  • に 表示具体时间

  • に 表示说明事物性质或状态时比较的基准或对象

  • で 表示范围,相当于"在..."

  • まで 表示终点,相当于"到...(为止)"
