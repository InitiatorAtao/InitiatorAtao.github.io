句型一览

名词/形容动词句

  • 现肯: は...だ（です）

  • 现否: は...ではない（ではありません）

  • 过肯: は...だった（でした）

  • 过否: は...ではなかった（ではありませんでした）

  • 推量: は...だろう（でしょう）

  • 中顿: は...で

  • 疑问: は...ですか

形容词句

  • 现肯: は...い（いです）

  • 现否: は...くない（くありません/くないです）

  • 过肯: は...かった（かったです）

  • 过否: は...くなかった（くありませんでした/くなかったです）

  • 推量: は...かろう（でしょう）

动词句

  • 现肯: は...行く（行きます）

  • 现否: は...行きません

  • 过肯: は...行った（行きました）

  • 过否: は...行きませんでした

  • 邀请: は...ませんか/ましょうか/ましょう

存在句

两种开头:

  • 某处に某物が

  • 某物は某处に

  • 现肯: ある/いる（あります/います）

  • 现否: ない/いない（ありません/いません）

比较句

  • A 比 B 更...: AはBより...です

  • 比起 B,A 更...: BよりAのほうが...です

  • A 不如 B...: AはBほど...ではない

  • 在 C 中比 A 更...的东西没有了: CではAほど(...之物)はない

愿望

  • A想要B: AはBが欲しい（です）

  • 想要做某事: 动词ます形和ます之间插入形容动词たい

动词て形

表示动作的先后,原因,方式,方法,拨音便和が行い音便为で行

  • 正在...: ...でいます

  • ...完成后: ...でから

  • 请求: ...でください

变形

形容词

连体:い 连用:く 中顿:くて 假定:ければ 推量:かろう

形容动词

连体:な 连用:に 中顿:で 假定:なら（ば） 推量:だろう

助词一览

  • は 主语

  • の 的

  • か 疑问

  • に

    存在场所 具体时间 基准 比例 对象 移动目的地 变化结果 动作的目的 归着点 变化后果

  • が

    小主语 表转折

  • と

    和

  • や

    一类

  • も 也

  • より 比

  • で

    范围 动作场所 动作方法,手段

  • から 起点

  • まで 终点

  • を 宾语

  • ね 确认,叮嘱,感叹

  • よ 判断,主张,提醒

  • へ 动作方向

  • ので 因果关系

  • として 身份,资格,立场

  • あまり 不太...

  • くらい(ぐらい) 表示大概

  • 欲しい(ほしい) 想要...
