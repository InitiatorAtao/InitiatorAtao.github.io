生成与导出

支持生成 .msi 和 .exe 格式的 Windows 安装包,也可以生成 Android apk 包.

  1. 构建发布版本: cargo tauri build

  2. 构建 Android apk 包: cargo tauri android build –apk

    构建的包是没有签名的版本,需要生成电子签名后才能安装.

    jarsigner -keystore [.jks] -signedjar [signed.apk] [unsigned.apk] [keyname]

    签名后需要进行对齐:

    zipalign -p 4 [signed.apk] [aligned.apk]

    其中 zipalign 是 Android SDK 的一个组件,位于类似

    C:/Users/[username]/AppData/Local/Android/Sdk/build-tools/[version]

    的目录下.
